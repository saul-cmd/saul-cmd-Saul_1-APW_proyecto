CSS (Cascading Style Sheets)

CSS es el lenguaje utilizado para estilizar el contenido de HTML. Permite aplicar diseño visual a las páginas web, incluyendo colores, fuentes, márgenes, y mucho más.

Sintaxis CSS

selector {
  propiedad: valor;
}

En CSS, un selector apunta a los elementos que se quieren estilizar, y cada propiedad afecta un aspecto del estilo de esos elementos.

Tipos de Selectores

  • Selector de etiqueta: Selecciona elementos por su nombre de etiqueta, por ejemplo, p, h1.
  • Selector de ID: Selecciona un elemento único usando #, por ejemplo, #miId.
  • Selector de clase: Selecciona elementos con una clase usando ., por ejemplo, .miClase.

Propiedades Comunes

A continuación se muestran algunas propiedades CSS utilizadas frecuentemente:

  • color: Define el color del texto.
  • background-color: Establece el color de fondo del elemento.
  • font-size: Define el tamaño de la fuente.
  • margin: Espacio exterior alrededor de un elemento.
  • padding: Espacio interior dentro de un elemento.
  • border: Define el borde alrededor de un elemento (grosor, tipo y color).

Pseudoclases

Las pseudoclases permiten definir un estilo especial para un estado específico de un elemento, como cuando el usuario coloca el puntero sobre él:

  • :hover - Aplica el estilo cuando el usuario pasa el ratón sobre el elemento.
  • :active - Aplica el estilo cuando el elemento es activado (ej. un botón clicado).
  • :focus - Aplica el estilo cuando el elemento recibe foco (como un campo de entrada).
button:hover {
  background-color: #2980b9;
}

Pseudoelementos

Los pseudoelementos permiten aplicar estilos a partes específicas de un elemento:

  • ::before - Inserta contenido antes del contenido del elemento.
  • ::after - Inserta contenido después del contenido del elemento.
  • ::first-line - Aplica estilo a la primera línea del texto en un elemento.
p::first-line {
  font-weight: bold;
}

Modelo de Caja

CSS utiliza el modelo de caja para definir el espacio de cada elemento en la página:

  • Margin: Espacio exterior de la caja.
  • Border: Borde que rodea la caja.
  • Padding: Espacio entre el contenido y el borde.
  • Content: Área donde se encuentra el contenido.

Ejemplo de un estilo utilizando el modelo de caja:

.miCaja {
  width: 300px;
  padding: 10px;
  border: 1px solid black;
  margin: 20px;
}

Ejemplo Completo de CSS

En este ejemplo, definimos varios estilos para diferentes elementos en una página:

body {
  font-family: Arial, sans-serif;
  background-color: #f9f9f9;
}

h1 {
  color: #2c3e50;
  text-align: center;
}

.boton {
  background-color: #3498db;
  color: white;
  padding: 10px 20px;
  border: none;
  border-radius: 5px;
  cursor: pointer;
}

.boton:hover {
  background-color: #2980b9;
}
