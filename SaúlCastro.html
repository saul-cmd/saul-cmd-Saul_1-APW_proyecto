CSS (Cascading Style Sheets)

CSS es el lenguaje utilizado para estilizar el contenido de HTML, permitiendo aplicar diseño visual como colores, fuentes y márgenes.

Sintaxis CSS

selector {
  propiedad: valor;
}

En CSS, un selector apunta a los elementos que queremos estilizar, y cada propiedad afecta un aspecto del estilo de esos elementos.

Tipos de Selectores

  • Selector de etiqueta: Selecciona elementos por su nombre, como p o h1.
  • Selector de ID: Selecciona un elemento único usando #, como #miId.
  • Selector de clase: Selecciona elementos con una clase usando ., como .miClase.

Propiedades Comunes

  • color: Define el color del texto.
  • background-color: Establece el color de fondo.
  • font-size: Tamaño de la fuente.
  • margin: Espacio exterior alrededor del elemento.
  • padding: Espacio interior dentro del elemento.
  • border: Define el borde alrededor del elemento.

Pseudoclases

  • :hover - Aplica estilo cuando el usuario pasa el ratón sobre el elemento.
  • :active - Aplica estilo cuando el elemento es activado (como un botón clicado).
  • :focus - Aplica estilo cuando el elemento recibe foco (como en un campo de entrada).
button:hover {
  background-color: #2980b9;
}

Pseudoelementos

  • ::before - Inserta contenido antes del contenido del elemento.
  • ::after - Inserta contenido después del contenido del elemento.
  • ::first-line - Aplica estilo a la primera línea del texto.
p::first-line {
  font-weight: bold;
}

Modelo de Caja

  • Margin: Espacio exterior de la caja.
  • Border: Borde que rodea la caja.
  • Padding: Espacio entre el contenido y el borde.
  • Content: Área donde se encuentra el contenido.

Ejemplo Completo de CSS

body {
  font-family: Arial, sans-serif;
  background-color: #f9f9f9;
}

h1 {
  color: #2c3e50;
  text-align: center;
}

.boton {
  background-color: #3498db;
  color: white;
  padding: 10px 20px;
  border: none;
  border-radius: 5px;
  cursor: pointer;
}

.boton:hover {
  background-color: #2980b9;
}
